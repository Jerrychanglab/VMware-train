  - ## 虛擬化的定義
    ### 了解VMware之前，這邊來探討何謂[虛擬化]，早期實體機的時代，一台機器安裝一個OS系統提供服務，資源的應用上會出現浪費的現象。虛擬化技術由這而延伸出在實體層上，模擬多個可完整獨立的硬體資源，使資源利用率最大化的一個技術。
